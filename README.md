## Comment compiler and lancer le programme sur LINUX et MAC avec "make"

> Dans le répertoire parent | require gcc

```batch
make compile
```

```batch
make debug
```

## Comment compiler et lancer le programme sur WINDOWS

> Dans le répertoire parent | require gcc

```batch
gcc main.c -o <output name file>
```
