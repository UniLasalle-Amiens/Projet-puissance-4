## How to compile and debug on LINUX and MAC with "make"

> On the main repository | require gcc

```batch
make compile
```

```batch
make debug
```

## How to compile and debug on WINDOWS

> On the main repository | require gcc

```batch
gcc main.c -o <output name file>
```
